Introducing PixelLines Custom ROM

Posted on 05 February 2026

Welcome to PixelLines.

PixelLines is a Custom ROM project focused on delivering a simple, stable, and user-friendly system. Currently based on the Android 16 (QPR0) platform, this project aims to provide a refined experience that balances performance with the latest system innovations.

The project is currently under private development as we refine the core features and optimize system stability. We are actively working to expand device support and ensure a smooth experience for the community once the development phase is complete.

PixelLines is open for collaboration, and we welcome developers interested in joining as official maintainers or contributing through device ports. We value community feedback and input to help shape the future of this project.

Thank you for your support of PixelLines.
